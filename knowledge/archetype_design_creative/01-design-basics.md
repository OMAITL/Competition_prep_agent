# 设计与创意类竞赛备赛要点（赛道通用）

## 常见赛制

- 广告艺术、数字艺术、工业设计等：命题或自由选题，提交作品 + 设计说明。
- 常分平面 / 视频 / 交互 / 产品等赛道，有尺寸、时长、文件格式限制。
- 评审侧重创意、美学、与命题契合度、完成度。

## 核心能力

- 设计调研与用户 / 场景理解
- 视觉表达：构图、色彩、 typography、动效（视媒介而定）
- 工具熟练度：Adobe 系列、Figma、Blender、AE 等（按赛项）
- 设计说明撰写：概念、过程、亮点、应用价值
- 版权与素材合规（字体、图片、音乐）

## 推荐学习顺序

1. 精读命题与评分标准，确定 1 个清晰概念方向
2. 草图 / 分镜 → 中期检查 → 终稿
3. 设计说明与展板 / 视频同步打磨
4. 导出规范检查（分辨率、色域、文件大小）

## 常见失误

- 偏离命题或组别要求
- 作品精美但设计说明薄弱
- 使用未授权素材
- 临期导出导致格式错误或画质压缩过度
