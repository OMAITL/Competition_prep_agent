# 机械与结构设计类竞赛备赛要点（赛道通用）

## 常见赛制

- 机械创新设计、结构大赛、成图大赛、三维数字化设计等。
- 提交物常含设计说明书、图纸 / 三维模型、实物或动画演示、答辩。
- 部分赛项强调轻量化、可靠性、可制造性与成本。

## 核心能力

- 机械原理、材料与加工工艺基础
- CAD / CAE：SolidWorks、Creo、ANSYS 等（按赛项要求）
- 工程图纸规范与公差标注
- 实物制作或 3D 打印、装配调试
- 设计计算书与创新点阐述

## 推荐学习顺序

1. 理解命题约束（载荷、尺寸、功能、成本）
2. 方案对比 → 详细设计 → 仿真 / 校核（若允许）
3. 出图 / 建模 → 加工或打印 → 试验
4. 完善说明书与答辩 PPT

## 常见失误

- 设计不可加工或未考虑标准件采购周期
- 图纸与模型 / 实物不一致
- 强度 / 刚度未校核，现场失效
- 说明书缺少设计计算与试验对比
