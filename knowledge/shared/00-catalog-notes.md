# 教育部认可的全国大学生学科竞赛（名录使用说明）

本知识库条目供RAG检索，帮助Agent理解名录来源与生成约束。

## 名录来源

- 清单来自中国高等教育学会《2025 全国普通高校大学生竞赛分析报告》竞赛目录（84 项）。
- 产品内每条竞赛包含：序号、正式名称、主办方官网链接（个别需人工补全）。

## 生成约束

- 赛项名称必须与名录一致，不得擅自改名。
- 无 verified 来源的 URL 不得编造；链接不确定时标注「待补充」。
- 备赛路径需考虑用户填写的截止日期与每周可投入小时数。
- 若知识库与官网摘要冲突，以官网为准，并在 warnings 中说明。

2. 补齐基础技能
