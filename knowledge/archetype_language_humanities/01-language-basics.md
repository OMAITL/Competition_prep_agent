# 语言与人文类竞赛备赛要点（赛道通用）

## 常见赛制

- 英语演讲、辩论、阅读 / 写作：常分校、省、国阶段，含定题 / 即兴环节。
- 经典诵读、跨文化等：作品录制或现场展示 + 文化阐释。
- 评分侧重语言准确度、内容逻辑、舞台表现与文化理解。

## 核心能力

- 听说读写：词汇、语法、发音、修辞与篇章结构
- 即兴组织与限时表达
- 文化背景与议题资料积累
- 舞台表现：声音、肢体语言、时间控制
- 备赛稿打磨与脱稿 / 提词平衡

## 推荐学习顺序

1. 研读赛项章程与样题 / 往届题目
2. 每天固定输出（录音 / 录像复盘）
3. 模拟评委 Q&A 或即兴题
4. 赛前调整稿长与高潮段落，确保不超时

## 常见失误

- 背稿痕迹重、与提问脱节
- 超时或内容不足
- 发音 / 语法错误反复出现未纠正
- 忽视文化题的背景准备
