# 电子与嵌入式类竞赛备赛要点（赛道通用）

## 常见赛制

- 全国大学生电子设计竞赛：3 天 4 夜现场命题，设计 + 制作 + 测试。
- 嵌入式、集成电路、光电设计等：作品 + 文档 + 答辩，周期较长。
- 信息安全类：CTF 或攻防演练 + 报告，规则以当年赛题为准。

## 核心能力

- 电路设计、PCB 或面包板调试、信号与电源完整性基础
- 嵌入式 C / 驱动、RTOS、通信协议
- 仪器使用：示波器、逻辑分析仪、万用表
- 系统联调与故障定位
- 设计报告：方案对比、测试数据、实物照片

## 推荐学习顺序

1. 巩固模电 / 数电 / 单片机基础
2. 完成 2～3 个完整小作品（传感、通信、控制）
3. 练习限时方案设计与器件选型
4. 模板化设计报告与测试表格

## 常见失误

- 方案过大，时间内无法调通
- 无备用模块，现场故障无法恢复
- 测试数据缺失或与原理图 / 代码不一致
- 忽视电源、接地与干扰导致不稳定
