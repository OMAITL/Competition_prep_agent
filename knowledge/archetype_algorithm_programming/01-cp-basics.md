# 算法与程序设计竞赛备赛要点（赛道通用）

## 常见赛制

- ACM-ICPC / CCPC：多人组队，若干小时若干题，通过 OJ 提交，罚时排名。
- 蓝桥杯：个人赛为主，客观题 + 编程题，省赛 / 国赛分级。
- 软件类程序设计赛：常含开发题、算法题或作品展示，以当年章程为准。

## 核心能力

- 基础算法：排序、搜索、贪心、动态规划、图论、数据结构
- 编码实现：C++ / Java / Python 熟练度与标准库使用
- 读题与建模：将题意转化为可编程逻辑，边界与复杂度分析
- 调试与对拍：样例、暴力、数据生成器
- 团队协作（团体赛）：分工读题、主写与交叉验证

## 推荐学习顺序

1. 选定主语言，巩固语法与 IO
2. 按专题刷题（洛谷、LeetCode、Codeforces 等），每专题 10～20 题
3. 参加 AtCoder / CF 虚拟赛或校赛练手
4. 赛前 2 周以历年真题限时模拟

## 常见失误

- 未读清数据范围导致 TLE / MLE
- 忽略 long long 与浮点精度
- 只刷题不总结模板与错题
- 团体赛沟通不足，重复造轮子或漏题
