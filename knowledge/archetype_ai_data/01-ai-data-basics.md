# 人工智能与数据类竞赛备赛要点（赛道通用）

## 常见赛制

- 算法精英赛 / 数据挖掘赛：给定数据集与评价指标，提交预测结果或模型。
- 企业命题赛：结合业务场景，常含报告 + 代码 + 答辩。
- 时长从数小时线上赛到数周迭代赛不等，需看清提交次数与榜单规则。

## 核心能力

- Python 与常用库：NumPy、Pandas、scikit-learn、PyTorch / TensorFlow
- 特征工程、模型选择与调参、集成学习
- 评价指标理解（AUC、F1、mAP、RMSE 等）与过拟合控制
- 可复现：随机种子、环境说明、推理脚本
- 技术报告：方法、实验对比、误差分析

## 推荐学习顺序

1. EDA 与 baseline（简单模型 + 官方 metric）
2. 特征与模型迭代，记录实验表格
3. 集成或后处理冲榜（注意过拟合与公榜 / 私榜差异）
4. 整理代码与简短说明文档

## 常见失误

- 未对齐评价指标或提交格式
- 数据泄漏（用测试信息训练）
- 只追榜不写字段说明，答辩 / 报告阶段吃亏
- 环境不可复现，复评失败
