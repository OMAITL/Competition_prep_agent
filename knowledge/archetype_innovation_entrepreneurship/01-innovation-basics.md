# 创新创业类竞赛备赛要点（赛道通用）

## 常见赛制

- 「互联网+」、挑战杯等：商业计划书 + 路演 + 答辩，常分创意组 / 创业组。
- 服务外包、iCAN 等：强调应用落地、市场与交付能力。
- 校赛 → 省赛 → 国赛逐级晋级，材料版本需与注册信息一致。

## 核心能力

- 问题洞察与用户需求验证
- 商业模式、市场分析、竞品对比
- 产品 / 技术方案的可行性说明（不必过度堆砌技术）
- 路演 PPT 与 5～8 分钟陈述
- 答辩应对：财务、壁垒、团队、社会价值

## 推荐学习顺序

1. 确定痛点与目标用户，完成最小验证（访谈 / 问卷 / MVP）
2. 撰写 BP 骨架：痛点、方案、市场、模式、团队、财务
3. 制作路演 PPT，校内预答辩 ≥ 2 轮
4. 按官方模板统一格式与附件清单

## 常见失误

- 技术描述过多、商业逻辑不清
- 数据无来源或夸大市场规模
- PPT 文字过密，超时
- 团队成员分工与股权 / 贡献无法在答辩中说清
